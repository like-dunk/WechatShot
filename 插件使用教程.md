批量截图助手使用教程

这个插件主要有 4 个功能：
1. 批量截图（四平台）
把 Excel 或 CSV 里的视频号、头条号、抖音、小红书链接批量打开并截图。
2. 补缺失截图
如果之前有些图片没截成功，可以只补缺失的图片。
3. 生成 PPT
可以用刚截好的图片自动生成 PPT，也可以用已有截图生成 PPT。
4. Excel 纠错 / 修正
可以检查 Excel 里的发布时间、标题、平台、账号、链接等信息。
安装插件
1. 打开 Chrome 浏览器。
2. 在地址栏输入：chrome://extensions/
3. 打开右上角的“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择插件文件夹：shipinhao_chrome_extension
6. 安装完成后，点击 Chrome 右上角的插件图标。
7. 打开“批量截图助手”。
批量截图
1. 打开插件。
2. 点击“选择 Excel / CSV 文件”。
3. 选择你的任务表。
4. 在“处理方式”里选择：完整截图：处理表格中的全部任务
6. 看一下“任务预览”。确认序号、昵称、文件名、链接没有问题。
7. 点击“开始截图”。
8. 截图过程中，不要关闭插件打开的 Chrome 窗口。
9. 等待任务完成。
10. 一般路径是：下载 / 截图 / 时间_任务表名_截图；同时每次运行都会自动新建一个文件夹
只补缺失截图
适合场景：之前已经截过一部分，只想补没有截到的图片。
1. 打开插件。
2. 点击“选择 Excel / CSV 文件”。
3. 选择原来的任务表。
4. 在“处理方式”里选择：精准补充：只补缺失并下载修复版 ZIP
5. 选择或拖拽之前已经下载好的截图文件夹。
6. 插件会自动判断哪些图片缺失。
7. 点击“开始截图”。
8. 完成后，会自动下载一个修复版 ZIP。
9. 修复版 ZIP 里包含原来的图片和新补的图片。
截图完成后自动生成 PPT
默认情况下，插件已经勾选：截图完成后同时生成当前 PPT 模式
点击开始截图后会自动生成 PPT：
1. 先在“用截图生成 PPT”里选择 PPT 模式。
2. 保持“截图完成后同时生成当前 PPT 模式”勾选。
3. 点击“开始截图”。
4. 截图完成后，插件会自动生成并下载 PPT。
PPT 模式说明：
1. 发布剪报多图铺页：一页放多张截图。
2. 链接截图单图单页：一张截图生成一页。
3. 发布信息截图单图单页：一张截图生成一页，并带发布账号、平台、标题、链接、时间。
用已有截图生成 PPT
适合场景：已经有截图，不想重新截图，只想生成 PPT。
1. 打开插件。
2. 找到“用截图生成 PPT”。
3. 选择 PPT 模式。
4. 点击“选择 ZIP”或“选择文件夹”。
5. 选择已有截图压缩包，或者选择已有截图文件夹。
6. 点击“生成并下载 PPT”。
7. 等待 PPT 自动下载。
Excel 纠错 / 修正
适合场景：检查表格里的发布日期、发布标题、发布平台、发布账号、链接是否正确，以及标题主题是否异常（如链接到了竞品或无关内容）。
1. 先导入 Excel 或 CSV 文件。
2. 如果是第一次使用，点击底部的”内部配置”。
3. 填写 API Token。
4. 点击”Excel 纠错”。
5. 插件会下载一个带颜色标记的检查结果。
6. 如果想让插件自动修正能确认的信息，点击”Excel 修正”。能查到真实信息的行（包括重复链接的行）都会被回填成正确值；只有链接失效、查不到信息的红色行不会被改动。
7. 标题异常检测（橙色标记）由后端自动完成，无需额外配置。
颜色说明：
红色：链接有问题，或者查不到信息。
紫色：链接重复（多行指向同一条内容；点“Excel 修正”时这些行的字段也会按查到的真实信息回填，但行仍标紫提醒你核对该保留哪一条）。
蓝色：账号重复（按查到的正确账号判定，所以即使原表里账号写错，修正后相同账号的行也会一起标蓝）。
黄色：表格内容和查到的信息不一致。
橙色：标题主题异常（例如链接到了竞品或无关内容），由后端自动检测。
